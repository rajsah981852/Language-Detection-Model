# Language-Detection-Model
Developed an end-to-end Language Detection System using Natural Language Processing and Machine Learning techniques.
The system classifies text into multiple languages including English, Russian, Thai, and Turkish.

*Key Highlights

Text preprocessing (Tokenization, Stopword Removal, Stemming)

Feature Engineering using Bag of Words (CountVectorizer)

Model Training using Multinomial Naive Bayes

Performance Evaluation using Accuracy Score & Confusion Matrix

Visualization using Seaborn Heatmaps

Model Export using Joblib for real-time predictions

*Tech Stack

Python

Pandas & NumPy

NLTK

Scikit-learn

Matplotlib & Seaborn
